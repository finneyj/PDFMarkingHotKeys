# PDFMarkingHotKeys

| Key | Action |
| ----------- | ----------- |
| ctrl-m | Turn on/of marking mode |
| `` ` `` | Record half-mark |
| 0-9 | record mark (0-9 respectively) |
| F1-F4 | set question number (sticky) |
| \# | free text comment |
| ^i | Add "(irrelevant)" comment |
| ^v | Add "(vague)" comment |
| ^g | Add "(generous)" comment |
